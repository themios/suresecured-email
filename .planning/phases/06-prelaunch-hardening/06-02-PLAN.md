<objective>
Implement first-touch attribution and harden the Shopify commission chain so the initiating salesperson gets paid.

Purpose: Core business promise — rep who did outreach owns the sale.
Output: Attribution library, migration, webhook refactor, unified click tracking, audit trail.
</objective>

<task id="1" audit="PL-010">
Create `migrations/007_attribution.sql`:

```sql
ALTER TABLE leads ADD COLUMN IF NOT EXISTS
  attributed_salesperson_id INTEGER REFERENCES salespeople(id),
  attributed_at TIMESTAMPTZ,
  attribution_source VARCHAR(50),
  attribution_locked BOOLEAN DEFAULT false;

CREATE TABLE IF NOT EXISTS commission_events (
  id SERIAL PRIMARY KEY,
  order_id INTEGER REFERENCES orders(id),
  salesperson_id INTEGER,
  client_id INTEGER,
  resolution_path TEXT NOT NULL,
  sale_amount NUMERIC(10,2),
  commission_earned NUMERIC(10,2),
  created_at TIMESTAMPTZ DEFAULT NOW()
);
```

Wire in `src/db.js` initDb.
</task>
